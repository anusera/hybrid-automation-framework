# Hybrid Automation Framework

## Overview

This project is a Selenium Java TestNG automation framework built for web application testing.

Application Under Test:

* SauceDemo

## Technologies Used

* Java
* Selenium WebDriver
* TestNG
* Maven
* Git

## Project Structure

* src/main/java - Framework code
* src/main/resources - Configuration files
* src/test/java - Test scripts
* src/test/resources - Test data
